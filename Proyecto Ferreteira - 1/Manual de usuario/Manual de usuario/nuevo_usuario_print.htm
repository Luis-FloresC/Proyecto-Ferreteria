7.2. Nuevo usuario

 
Funcionalidad del módulo: Permite poder ingresar nuevos usuarios para que puedan ingresar al sistema, asignandoles un nombre de usuario y una constraseña para su acceso. Para acceder a esta venta click en el botón de nuevo usuario.
 
 
Instrucciones
Se muestran los datos que debe ingresar en los campos antes de poder hacer click a cualquiera de los botones que usted presione.
 
  • Botón "Aceptar"
  • Seleccionar el nombre del empleado en el espacio de "Empleado".
  • Ingrese el nombre de usuario para dicho empleado en el espacio "Nombre del Usuario"
  • Escribir una contraseña (al menos 8 caracteres) para este nuevo usuario en el espacio "Contraseña"
  • Confirmar al contraseña escribiendola nuevamente en el espacio "Confirmar Contraseña".
  • Si desea guardar los cambios, presione “Aceptar”
     
  • Botón "Cancelar"
  • Sino desea realizar los cambios presione "Cancelar"  para descartarlos.
