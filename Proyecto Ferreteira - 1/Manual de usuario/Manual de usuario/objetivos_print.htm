3. Objetivos

 
  • Mostrar al usuario cómo usar de manera correcta el sistema.
  • Hacer que el usario sea capaz de manejar sin ningún problema cada una de las opciones, módulos que están dentro del sistema.
  • Permitir al usario conocer con exactitud el manejo de todas las áreas de la empresa por medio del sistema.
  • Lograr que el usario pueda usar de manera eficiente el sistema para los logros y metas de la empresa, que pueda dar a la empresa información de apoyo y refuerzo para la empresa.
