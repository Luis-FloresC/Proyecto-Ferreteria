5. Iniciar Sesión

 
Funcionalidad del módulo: Ha sido creado para que el usuario pueda iniciar sesión, debe ingresar valga la redundancia su usuario y contraseña, el cual se creó desde el momento en que fue registrado como empleado en la base de datos.
 
 
Instrucciones
 
  • Botón "Aceptar"
  • Ingresar el nombre de usuario en la primera caja de texto
  • Ingresar la contraseña de la cuenta de ese usario en la segunda caja de texto
  • Presionar el botón de "Aceptar" para acceder al sistema
